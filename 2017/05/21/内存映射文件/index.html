内存映射文件

发表于 2017-05-21 | 分类于 Java

内存映射文件：利用虚拟内存实现将文件“映射”到内存中。这样这个文件就可以当做是一个内存数组一样的访问，这比传统的文件操作要快得多。

映射：硬盘上文件的位置与进程逻辑地址空间中一块大小相同的区域之间的一一对应

1. java使用内存映射

步骤：

  • 引入java.nio包

  • 从文件中获得一个通道(channel)。

  • 通过调用FileChannel类的map方法从这个通道中获得一个ByteBuffer。其中，映射文件区域与映射模式支持三种方式：

    1.1. – FileChannel.MapMode.READ_ONLY:缓冲区只读

    FileChannel.MapMode.READ_WRITE：可读写。任何缓冲区的修改都会写回文件（非立即）
    – FileChannel.MapMode.PRIVATE：缓冲区可写，但修改不会传播到文件中
