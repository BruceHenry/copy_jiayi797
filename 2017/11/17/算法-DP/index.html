2. 什么时候用DP

三个条件满足其一即有可能是要用DP：

  • 求极值
  • 判断是否可行
  • 统计方案个数
  • 擅长将$2^n->n^2$

3. 什么时候不用DP

三个条件满足其一即有可能是不用DP：

  • 求所有具体的方案（输出所有回文串）
  • 输入是集合，而不是序列（求最长连续序列）
  • 暴力方法的复杂度已经是多项式级别，不擅长$n^3->n^2$

4. DP四要素

状态：f[][]的含义，最难！

方程：状态之间的联系，怎么用小状态算大状态

初始化：最小状态是什么，起点

答案：最大状态是什么，终点

VS递归三要素

  • 定义（状态）
    • 接受了什么参数
    • 做了什么事情
    • 返回了什么值
  • 拆解（方程）
    • 符合将参数变小
  • 出口（初始化）
    • 什么时候可以直接return
